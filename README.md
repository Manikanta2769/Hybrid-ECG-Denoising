# Hybrid-ECG-Denoising
A Python implementation of hybrid ECG signal denoising using Fourier Decomposition Method (FDM), Wavelet Transform, and Empirical Mode Decomposition (EMD). Evaluated on the MIT-BIH Arrhythmia Database and NSTDB using SNR, PRD, and MSE metrics. This repository contains an implementation of a hybrid ECG signal denoising framework inspired by the Fourier Decomposition Method (FDM). The pipeline combines FDM with Wavelet Transform and Empirical Mode Decomposition (EMD) to suppress baseline wander, power-line interference, white Gaussian noise, motion artifacts, and EMG noise while preserving ECG morphology. Performance is evaluated using Signal-to-Noise Ratio (SNR), Percentage Root Mean Square Difference (PRD), and Mean Squared Error (MSE) on MIT-BIH Arrhythmia Database and Noise Stress Test Database (NSTDB).
